CARACTERÍSTICAS DO PAINEL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ Painel expandível/colapsável (clique no botão 📅 Datas)

✓ Radio buttons para selecionar tipo de filtro:
  - Data de Emissão (quando o boleto foi criado)
  - Data de Vencimento (quando o boleto vence)

✓ Campos de data HTML5 (abre calendário ao clicar):
  - De: data inicial
  - Até: data final

✓ Cores & Feedback:
  - Botão INATIVO: cinza (#1a1a1a)
  - Botão ATIVO: azul (#1a5490)
  - Painel: fundo escuro (#0f0f0f)
  - Inputs: fundo preto (#111111)

✓ Auto-limpeza: ao trocar de filtro (emissão → vencimento),
  os campos anteriores são automaticamente zerados

✓ Compatível com:
  - Busca por texto
  - Filtro de status
  - Seleção de linhas (checkboxes)

✓ Atualização em tempo real:
  - A tabela se atualiza conforme você muda as datas
  - Sem necessidade de recarregar a página


FLUXO DE USO RECOMENDADO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Abra o painel clicando em "📅 Datas"

2. Escolha o tipo de data:
   • "Data de Emissão" - para ver quando foram criados
   • "Data de Vencimento" - para ver quando vencem

3. Defina o intervalo:
   • Clique em "De:" e selecione a data inicial
   • Clique em "Até:" e selecione a data final

4. Observe a tabela se atualizar em tempo real

5. Para limpar, clique em "Limpar datas"


ATALHOS & DICAS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 Combinar filtros:
   Busca: "Cliente A" + Status: "Pendente" + Datas: "Mai/2026"
   → Só mostra boletos pendentes desse cliente emitidos em maio

💡 Apenas data inicial:
   De: 2026-05-01 | Até: (em branco)
   → Mostra tudo a partir de 1º de maio

💡 Apenas data final:
   De: (em branco) | Até: 2026-05-31
   → Mostra tudo até 31 de maio

💡 Período específico:
   De: 2026-05-15 | Até: 2026-05-25
   → Mostra apenas boletos desse intervalo (11 dias)
