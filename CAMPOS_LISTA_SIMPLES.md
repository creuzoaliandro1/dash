# 📋 Lista de Campos para Importação

SACADO_NOME: Nome do cliente/pagador (obrigatório)

VALOR: Valor do boleto em números (obrigatório, deve ser > 0)

NUM_TITULO: Número do documento/titulo do boleto

VENCIMENTO: Data de vencimento do boleto (formato DD/MM/YYYY)

EMISSAO: Data de emissão do boleto (formato DD/MM/YYYY)

SACADO_CIC: CPF ou CNPJ do cliente/pagador

SACADO_ENDERECO: Endereço/Logradouro do cliente

SACADO_NUMERO: Número do endereço do cliente

SACADO_BAIRRO: Bairro do cliente

SACADO_CIDADE: Cidade/Município do cliente

SACADO_UF: Estado/Unidade Federativa do cliente (2 caracteres)

SACADO_CEP: CEP do cliente

SACADO_TELEFONE: Telefone do cliente

SACADO_EMAIL: Email do cliente

NOSSO_NUMERO: Número identificador do banco para o boleto

CODIGO_BARRAS: Linha digitável/código de barras do boleto

AVALISTA_NOME: Nome da pessoa/empresa avalista

AVALISTA_CIC: CPF ou CNPJ do avalista

VALOR_PAGAMENTO: Valor que foi pago (se boleto já foi pago)

DATA_PAGAMENTO: Data em que o boleto foi pago (formato DD/MM/YYYY)

STATUS: Situação do boleto (pendente, pago, atrasado, cancelado)

DESCRICAO: Descrição ou observação sobre o boleto
